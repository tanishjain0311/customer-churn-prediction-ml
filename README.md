# Customer Churn Prediction using Machine Learning

## Overview

Customer churn is one of the biggest challenges faced by businesses today. This project focuses on building a robust Machine Learning model to predict whether a customer is likely to leave (churn) or stay with the company.

The solution helps businesses take proactive actions to improve customer retention and reduce revenue loss.

---

## Objectives

* Identify key factors influencing customer churn
* Perform detailed Exploratory Data Analysis (EDA)
* Build and evaluate Machine Learning models
* Predict customer churn with high accuracy

---

##  Project Workflow

### 1️ Data Collection

* Dataset containing customer demographics, account details, and usage patterns

### 2️⃣ Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature scaling

### 3️⃣ Exploratory Data Analysis (EDA)

* Visualizing churn distribution
* Identifying important features affecting churn

### 4️⃣ Model Building

* Applied Machine Learning algorithms:

  * Logistic Regression
  * Decision Tree
  * Random Forest (if used)

### 5️⃣ Model Evaluation

* Accuracy Score
* Confusion Matrix
* Precision, Recall, F1-score
